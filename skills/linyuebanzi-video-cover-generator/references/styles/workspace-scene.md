# 工作台场景风

> 本风格模板须与 `references/brand_baseline.md` 中的品牌基线一起使用。

把人物放进真实的 desk/setup 场景里，像 Vlog 截图，适合设备评测、真实场景演示。

### 视觉特征

- 场景：真实的工作台 / 书房 / 工作室
- 人物：坐在或站在工作台旁，占比 30-40%
- 道具：显示器、键盘、笔记本、咖啡杯、植物等
- 光线：自然光或暖台灯，营造真实氛围
- 标题：画面上方或留白处，不要遮挡人脸

### 固定

- 场景必须真实可信，避免纯模板背景
- 人物：参考图 1 五官，半身或大半身
- 水印：@林月半子 🦞

### 变量

- 场景类型（home office / 工作室 / 书房）
- 桌上道具
- 光线方向（窗边自然光 / 台灯暖光）
- 标题位置
