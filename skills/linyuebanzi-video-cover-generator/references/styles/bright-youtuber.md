# 明亮 YouTuber 风

> 本风格模板须与 `references/brand_baseline.md` 中的品牌基线一起使用。

背景明亮温暖，人物更生活化，整体像海外科技 YouTuber 的封面，适合想"破圈"或降低技术距离感的内容。

### 视觉特征

- 背景：暖米色 / 浅灰 / 奶油色渐变，干净明亮，带柔和环境光
- 标题：粗黑体 + 重白色描边，颜色可用黄、绿、橙等高对比色
- 人物：占比 30-35%，居中偏下，更放松自然
- UI 元素：截图清晰、白底或浅色，做问题/解决对比
- 整体：明亮、亲和、高能量

### 固定

- 人物：参考图 1 五官，半身，可穿深色或午夜绿 polo
- 表情：自信微笑看镜头（不夸张）
- 水印：@林月半子 🦞
- 元素不超出安全区

### 变量

- 背景暖色具体色调（米白 / 暖灰 / 奶油 / 浅蓝灰）
- 标题描边颜色
- 人物衣着颜色
- UI 截图风格（白底 / 暗色）
